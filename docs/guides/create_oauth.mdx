---
title: "Add OAuth app"
---

You can add your own OAuth app. In order to obtain the Client ID and Client Secret, it's necessary for you to register your application with the API provider. It's also important to register the required scopes with both the API provider and on the OAuth page.

## Create a New OAuth Account

1. On the OAuth tab, Click on `+ New OAuth App`:

<Frame>
  <img src="/images/oauth/create_oauth.png" />
</Frame>

2. Choose the integration you want to connect and provide the necessary information.

<Frame>
  <img src="/images/oauth/sample_oauth.png" />
</Frame>

3. Enter the client_id, client_secret and scopes according to the integration. Once done you can click on create.

4. Once the OAuth is created it will show in the OAuth page. You can now access repective OAuth in the link page.

## Callback URL

Please add this Callback URL respectively in the OAuth app

for local, `http://localhost:8000/api/v1/oauth/callback`

for self-hosted `https://{your-domain}/api/v1/oauth/callback`
